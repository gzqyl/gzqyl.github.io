Notes: if streaming media mode turned off (default status), user could play the video by webview (it may use native video player or not which controlled by website developer), but if streaming media mode turned on, once the app found m3u8 url request, it will play the m3u8 link by native video player always.

You could find the difference when mode is on, click the button to play will be forced to use native player with a modal view.
